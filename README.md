# 💧 ペットボトル噴流の吹上げ高さシミュレータ

Streamlitで構築した簡易流体モデルにより、ペットボトル内圧と吹上げ高さの関係を可視化します。

### 🌐 公開方法
1. このリポジトリをGitHubにアップロード
2. [Streamlit Cloud](https://share.streamlit.io) にログイン
3. 「New App」 → このリポジトリを選択
4. 「bottle_water_jet.py」を指定
5. 🚀 **Deploy**

### 📘 モデル概要
- 水の密度: ρ = 1000 kg/m³  
- 縮流係数: Cd = 0.62  
- 重力加速度: g = 9.81 m/s²  
- 効率 η、外周流速度比 r により減衰補正# bottle-water-jet
